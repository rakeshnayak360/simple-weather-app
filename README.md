# simple-weather-app
Created a simple weather App using ReactJS, Styled Component and third-party weather API.

<a href="https://rakeshnayak360.github.io/simple-weather-app/build/">Live demo</a>

### Installation
- Clone the repo
- Navigate to the folder
- Run npm install
- Run npm start
